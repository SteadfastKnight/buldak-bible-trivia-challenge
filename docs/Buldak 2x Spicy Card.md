# 🔥 Buldak Trivia Biblică — Card Jucător
## 2x Picant (Multiplicator 3x)

### Fiecare Rundă
1. Mănâncă înghițitura ta 🔥
2. Alege opțional o întrebare (sau pasează — fără penalizare)
3. Folosește alinare dacă e nevoie (costă puncte!)

### Punctaj (Totul ×3)
| Acțiune | Puncte |
|---|---|
| Ușor ✅ | +300 |
| Mediu ✅ | +750 |
| Greu ✅ | +1.500 |
| Ușor ❌ | −150 |
| Mediu ❌ | −300 |
| Greu ❌ | −600 |
| Apă 💧 | −225 |
| Lapte 🥛 | −450 |
| Supraviețuiești o rundă | +50 |

### Ține Minte
- Poți **pasa** orice întrebare — fără penalizare
- Poți **abandona** oricând — păstrezi punctele câștigate până atunci
- Scorurile sunt **ascunse** până la final
- Clasamentul se dezvăluie după **Runda 4**

*Ai ales durerea. Asumă-ți-o. 🔥📖*
